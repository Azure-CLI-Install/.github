## Fixing Azure CLI Setup Problems

- Azure CLI not recognized after installation? Restart the terminal, verify PATH settings, and review azure cli install notes for your platform.  
- Sign-in failing with azure login or az login? Confirm the correct account, tenant, browser session, and device-code flow when needed.  
- Commands running against the wrong subscription? Use az account list and az account set before continuing with azure cli commands.  
- Extension errors appearing during azure devops cli work? Update Azure CLI, reinstall the extension, and check azure cli documentation for version requirements.  
- Scripts behaving differently on Windows? Compare shell quoting rules, confirm azure cli windows paths, and test the same az cli command interactively before automation.
